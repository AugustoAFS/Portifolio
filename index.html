01. Sobre Mim

Olá! Sou Augusto, um desenvolvedor fullstack apaixonado por criar soluções robustas e escaláveis. Minha jornada na programação começou com C# e .NET, e desde então venho expandindo meus conhecimentos para o ecossistema completo de desenvolvimento web.

Atualmente, foco em tecnologias como C#, .NET Core, Angular, e SQL Server. Acredito que a combinação de backend sólido com frontend moderno é a chave para aplicações de sucesso.
